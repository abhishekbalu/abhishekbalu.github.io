Hi, I am Abhishek Balasubramaniam

Graduate Research Assistant at Colorado State University

I am currently working on object detection model optimization techniques for low power perception with deep learning for emerging autonomous vehicles using Keras and TensorFlow. Experimenting with optimization techniques such as pruning, quantization, knowledge distillation, encoding and decoding, and co-optimization techniques for reducing the model size and memory allocation. We use Nvidia Jetson TX2 for testing and evaluation

Recreated the SSD object detection model on using TensorFlow and Keras (not using TF object detection API). Worked on mixed-precision pruning, iterative layer by layer pruning and quantization.
